# Signal

A cross-sectional, multi-factor quantitative signal engine for US stocks/ETFs and major crypto.

Dual-horizon: short-term (3–7 day) mean-reversion + momentum, and position (1 week–1 quarter)
trend/momentum. Ranks a universe, surfaces the highest-conviction long setups, and keeps an
**honest shadow ledger** — every signal's forward outcome tracked out-of-sample, nothing simulated
shown without a label.

Free-tier only. No paid services.
